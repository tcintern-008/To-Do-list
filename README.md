# 📝 Interactive To-Do List

A simple To-Do List web app built with **HTML, CSS, and Vanilla JavaScript** as part of my .NET backend internship — Day 6 task focused on revising JavaScript fundamentals (variables, functions, arrays/objects, loops, conditionals, DOM manipulation, and Local Storage).

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks persist after refresh using Local Storage

## Bonus Features
- Filter tasks (All / Completed / Pending)
- Edit existing tasks
- Live count of completed and pending tasks

## Tech Used
- HTML5
- CSS3 (dark theme UI)
- Vanilla JavaScript (ES6)

## What I Practiced
- Working with arrays of objects
- DOM manipulation and event listeners
- Storing and retrieving data using `localStorage`
- Basic conditional logic and loops

## How to Run
1. Clone this repo
2. Open `index.html` in your browser
3. Start adding tasks!
